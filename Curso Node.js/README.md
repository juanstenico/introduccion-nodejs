# Introducción a Node.js - Mi Primer Script

## Descripción del proyecto

Este proyecto fue desarrollado como parte del Módulo 1 - Unidad 1 de Introducción a Node.js.

El objetivo es crear un primer programa utilizando Node.js, mostrando un mensaje de presentación, información del sistema operativo mediante el módulo nativo `os` y un mensaje personalizado utilizando un módulo propio.

## Archivos del proyecto

- app.js
- mensaje.js
- README.md

## Cómo clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

Ingresar a la carpeta del proyecto:

```bash
cd TU-REPOSITORIO
```

## Cómo ejecutar el proyecto

```bash
node app.js
```
